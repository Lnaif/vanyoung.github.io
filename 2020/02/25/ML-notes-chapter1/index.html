<Machine learning> -- Introduction

Introduction

Definition

The “experience” of people corresponds to the “data” in the computer, machine learning is just such a subject that the computer learn these experience data and generate an algorithm model to make effective judgement in new situation.

Mitchell, author of another classic text book, gives a formal definition that assumes:

  • P: the performance of a computer program on a tasks class T
  • T: the class of tasks that the computer program wants to implement
  • E: representing experience, that is the historical data sets

If the computer program gains an improvement in performance P on task T by using experience E, it is said that the program learned from E.

Terminology

Suppose we collect data on a batch of watermelons: (color = green;Root = curl;Beat=noise), (color = black;Root = slightly curled;Beat = dull), (color = light;Root = stiffening;Beat = crisp)…Inside each pair of parentheses is a record of a watermelon.

  • The set of all records——Data set

  • Each record——Instance or Sample

  • A single characteristic——Feature or Attribute

  • Each record represented on the coordinate axis by a point—— Feature vector

  • The number of features of a sample——Dimensionality

    When dimensions are very large, it becomes a ‘dimension disaster’

In the process of learning experience data and generation algorithm model, each record is called a “training sample”, and we hope to use new sample to test the effect of the pre-trained model, so each new sample is called a “test sample”.

  • The set of all training sample——Training set
  • The set of all test sample——Test set
  • The ability of the model to apply to new samples——Generalization

In the case of watermelon, we want the computer to learn the features of watermelon and train a decision model to judge whether a new watermelon is good or not. What we predict is that good melon and bad melon, which are discrete values. Similarly, there are projections of future population from historical population data, and population is a continuous value.

  • The problem where the predicted value is discrete——Classification

  • The problem where the predicted value is continuous——Regression

In the process of predicting, we have labeled all the melons in the training set of pre-known features, and learning machine summarized rule by learning the features of good or bad melons. But there is also no label information situation while we do not know the quality of watermelon in advance.

  • Training labeled data——Supervised learning
    • including Classification and Regression
  • Training un-labeled data——Unsupervised learning
    • including Clustering and Association Rule
